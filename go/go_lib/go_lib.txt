我的系统物理 CPU 核心数是 4 个，逻辑 CPU 核心数是 8 个，所谓物理 CPU 核心数指的是真正插在物理插槽上 CPU 的核心数，逻辑 CPU 核心数指的是结合 CPU 多核以及超线程技术得到的 CPU 核心数，最终核心数以逻辑 CPU 核心数为准。
Mac查看物理cpu和逻辑cpu 核心数
sysctl hw.physicalcpu
hw.physicalcpu: 2

sysctl hw.logicalcpu
hw.logicalcpu: 4
